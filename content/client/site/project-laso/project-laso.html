Laso Concepts project

Helped build a mobile responsive single page MVP, that provided a platform to connect healthcare institution with their patients and their supporting team members. It featured direct and group messaging to help nuture their community. It featured a thorough user onboarding process, multiple profile support, a notification system that included on-site and email messaging, and blogging system. Built an admininistrative content management system, providing an interface for the admin to manually edit or remove user generated content, information, and accounts.
