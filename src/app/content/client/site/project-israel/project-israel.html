Viajemos A Israel project

"Viajemos A Israel" is a Spanish web app, that translates to "Let's Travel to Israel". This app provides detailed information on an annual historic trip to Israel and surrounding countries, that includes a robust guided-tour schedule. The app is a mobile responsive single page application. It features a user-friendly application process using team-friendly Google Forms, allowing the users to apply for each trip with an easy interface.
